# Core API Service

This is the backend service for Project Nexus core platform (Epic 1). It uses FastAPI and exposes CRUD endpoints for buildings, floors, rooms, employees, assets, and work orders. The OpenAPI spec is automatically generated by FastAPI.

## Running

```bash
cd services/core_api
python -m venv venv
source venv/bin/activate  # or .\venv\Scripts\activate on Windows
pip install -r requirements.txt
uvicorn app.main:app --reload
```
