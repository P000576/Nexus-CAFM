# Core API Service

This is the backend service for Project Nexus core platform (Epic 1). It uses FastAPI and SQLAlchemy ORM with SQLite persistence, exposing CRUD endpoints for buildings, floors, rooms, employees, assets, and work orders. The OpenAPI spec is automatically generated by FastAPI.

## Architecture

- **Framework**: FastAPI
- **ORM**: SQLAlchemy 2.0
- **Database**: SQLite (file-based, no setup needed)
- **Models**: Pydantic for request/response validation, SQLAlchemy for persistence

## Running

```bash
cd services/core_api
python -m venv venv
source venv/bin/activate  # or .\venv\Scripts\activate on Windows
pip install -r requirements.txt
uvicorn app.main:app --reload
```

The API will be available at `http://localhost:8000` with auto-generated docs at `/docs`.

## Database

- Uses SQLite for local development/testing
- Database file: `nexus.db` (auto-created on first run)
- Schema is created automatically on startup
- Can be easily migrated to PostgreSQL/MySQL by changing `DATABASE_URL`
